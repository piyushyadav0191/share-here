# share-here
Shared is a comment system which you can use to add it in your blog or any website where you need comments, reviews etc.

## Technology Used
** Front-end**
- Nextjs + Typescript
- Zustand
- Tailwind Css

** Back-end **
- Nodejs + Express
- Mongodb
